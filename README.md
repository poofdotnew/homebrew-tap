# Homebrew Tap for Poof

Official [Homebrew](https://brew.sh) tap for [Poof](https://poof.new) CLI tools.

## Install

```bash
brew tap poofdotnew/tap
brew install poof
```

## Update

```bash
brew update
brew upgrade poof
```

## Formulas

| Formula | Description |
|---------|-------------|
| `poof`  | Build, deploy, and manage Solana dApps on poof.new |
